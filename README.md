# Full Stack Task Manager

A full-stack task management application built with:

- React
- FastAPI
- SQLite

## Features

- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- Filter tasks
- Responsive UI

## Tech Stack

Frontend:
- React
- JavaScript
- CSS
- React Toastify

Backend:
- FastAPI
- SQLModel
- SQLite
